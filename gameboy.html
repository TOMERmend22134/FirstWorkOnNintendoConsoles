Nintendo Game Boy

The Game Boy is an handheld game console developed by Nintendo. It was first released in Japan on April 21, 1989. Nintendo Game Boy is home to many famous games, such as Pokémon Red and Blue, Tetris, Pokémon Gold and more.

There are 3 models

There are 3 different models of the Game Boy:
Game Boy, Game Boy Color and Game Boy Advance.

Game Boy

The Game Boy is a 8-bit handheld game console developed by Nintendo. It was first released in Japan on April 21, 1989.

Game Boy Color

The Game Boy Color is a handheld game console which was released in Japan on October 21, 1998. It is the successor to the Game Boy.

Game Boy Advance

The Game Boy Advance is a handheld game console that was released in Japan on March 21, 2001. It is the successor to the Game Boy Color.
